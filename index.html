S-D4RT 几何预训练 · Streaming D4RT-Geo Architecture

将 D4RT 的 query-based geometry learning 迁移到自动驾驶多相机、LiDAR、ego pose 场景，面向流式 4D 几何表征与端到端任务预训练。

Geometry Pretraining
S-D4RT-Geo · 流式 4D 几何预训练整体架构
  • 在线因果推理主干：仅使用历史与当前 11V 图像、标定初值和 LIO 位姿初值，不看未来。
  • Calibration Refiner 输出残差修正后的相机内外参，并进入 observation lifter 与 streaming memory update。
  • Gated streaming memory 维护 ego-centric 4D memory，为 Ray-D4RT 点查询解码器提供时空几何上下文。
  • 查询解码器沿相机射线采样少量候选深度 token，通过 query-to-candidate attention 聚合深度分布并输出 ego-frame 3D point。
  • 训练监督以单帧 LiDAR 稀疏 metric anchor 为主，可选 same-time multi-view gsplat rendering 约束几何与外观一致性。
  • 使用 Codex 做 AI-assisted research prototyping，并围绕参数量、显存、吞吐、训练时长和推理代价做 profile-driven validation。
